# Qoder Wiki

Qoder 官方文档知识库，提供产品使用指南和技术参考。

## 使用方式

1. 首先查阅 [INDEX.md](INDEX.md) 获取文档索引和内容概览
2. 根据用户问题定位相关文档
3. 使用 Read 工具读取对应的 docs/ 目录下的文档
4. 如文档信息不足，再考虑联网搜索补充

## 文档结构

```
docs/
├── 快速入门/          # 产品概述、安装登录、快速开始
├── 用户指南/          # 智能会话、Quest Mode、NEXT、Repo Wiki
├── 上下文/            # 代码库索引、@ Mention、规则、记忆
├── 拓展能力/          # Skills、Commands、自定义智能体、MCP、Deeplinks
├── 配置/              # 快捷键、网络代理
├── 账户/              # 定价、Credits、账单
├── 事件/              # 推荐计划、限时优惠
└── 支持/              # FAQ、故障排查、MCP常见问题
```
