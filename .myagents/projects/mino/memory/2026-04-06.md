### 根因

## 飞书 API 突破

### 问题
之前飞书文档 API 设置公开链接一直报 `invalid param`。

### 原因
PATCH 格式错误——之前包了 `{"permission_public": {...}}` 外层，正确格式是不包外层。

### 正确格式
```bash
PATCH https://open.feishu.cn/open-apis/drive/v1/permissions/{doc_id}/public?type=docx
Body: {"link_share_entity": "anyone_readable", "external_access": true}
```

### link_share_entity 取值
`tenant_readable | tenant_editable | anyone_readable | anyone_editable | closed`

### 新规则（2026-04-05确立）
**所有上传飞书的文档，一律设置为公开链接（anyone_readable）。**
