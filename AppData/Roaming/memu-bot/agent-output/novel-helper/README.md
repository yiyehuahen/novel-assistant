# 📚 小说写作助手

一个纯辅助工具，帮助小说作者完成创作。

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| 🎲 随机情节 | 生成冲突、爱情、虐心、冒险、悬疑、转折等类型情节 |
| 🎲 随机名称 | 生成人名、地名、门派、武器、功法、法宝 |
| | 支持6种风格：玄幻、奇幻、轻小说、都市、科幻、历史 |
| | 支持选择字数：2字、3字、4字 |
| | 支持定头字（指定第一个字） |
| 📚 素材库 | 收藏灵感、好词好句，按类型分类（情节/名称/对话/描写/灵感） |
| 🌙 夜间模式 | 护眼模式切换 |
| 📱 PWA支持 | 可添加到手机主屏幕，离线使用 |

## 🚀 使用方法

### 本地运行
直接用浏览器打开 `index.html` 即可。

### 部署到 GitHub Pages
1. 创建 GitHub 仓库
2. 上传所有文件
3. 在 Settings → Pages 中启用 GitHub Pages
4. 选择 `main` 分支，点击 Save

## 📁 文件结构

```
novel-helper/
├── index.html      # 主页面
├── style.css       # 样式文件
├── app.js          # JavaScript 逻辑
├── manifest.json   # PWA 配置
├── sw.js           # Service Worker
└── README.md       # 说明文档
```

## 🎨 名称生成器 - 风格说明

| 风格 | 特点 | 适用类型 |
|------|------|---------|
| 玄幻 | 东方仙侠风格，云天风雷等 | 修仙、玄幻小说 |
| 奇幻 | 西式奇幻，阿尔贝鲁等 | 西幻、魔法小说 |
| 轻小说 | 日系风格，佐藤铃木等 | 校园、轻小说 |
| 都市 | 现代风格，李王张刘等 | 现代、商战小说 |
| 科幻 | 未来风格，艾文凯等 | 科幻、星际小说 |
| 历史 | 古代风格，长安洛阳等 | 历史、古风小说 |
